Octagon Agent

Public and Private market intelligence agent that optimally routes requests to appropriate specialized agents

Function: OCTAGON.AGENT(prompt, [format])
prompt is the question or prompt for the Octagon agent
[format] optional output format with possible values of 'table' (default), 'cell', or 'raw'.
Use 'table' for generating tabular data spanning multiple cells, 'cell' for a single cell value, and 'raw' for more verbose text-based content.
